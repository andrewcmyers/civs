How_it_works_para {
  en {
    Anyone may create a new CIVS poll, but only authorized voters will
    know about it. Voters and the poll supervisor must have e-mail and
    web access. When a poll is created, voters are e-mailed with a
    URL where they can vote.  Public polls may be created that do not
    require voters to have e-mail addresses; in this case, one vote is
    allowed per IP address.
  }
  de {
    Jeder kann eine neue CIVS-Umfrage erstellen, aber nur autorisierte
    Wähler werden davon erfahren. Die Wähler und der Wahlleiter müssen
    über E-Mail- und Internetzugriff verfügen. Wenn eine Umfrage
    erstellt wird, erhalten die Wähler eine E-Mail mit einer URL, unter
    der sie abstimmen können. Es können öffentliche Umfragen erstellt
    werden, bei denen die Wähler keine E-Mail-Adressen benötigen; in
    diesem Fall ist pro IP-Adresse eine Stimme zulässig.
  }
}

Why_rank_paras {
    en  {
      <p>With CIVS, voters rank their choices rather than
        just picking their one favorite choice.
        This process gives accurate results because it
        collects more information from voters. It also
        helps avoid vote splitting and spoilers.
      </p><p>
        Ranked-choice voting is used in real elections. For example,
        Australia uses Single Transferable Vote (STV, also known as IRV).
        However, the Condorcet methods supported by CIVS are better than IRV
        at identifying consensus candidates.
        IRV can elect a candidate even though a majority of
        voters would prefer someone else, and
        <a href="http://bolson.org/~bolson/2009/20090303_burlington_vt_mayor.html">
        it has in real elections</a>.
        Unlike most other Condorcet systems,
        CIVS also has a proportional representation mode.
      </p>
    }
    de {
       <p>Mit CIVS ordnen die Wähler ihre Entscheidungen, anstatt nur ihre
        eine Lieblingswahl auszuwählen. Dieser Prozess liefert genaue
        Ergebnisse, da mehr Informationen von den Wählern gesammelt
        werden. Es hilft auch, Stimmenteilung und Spoiler zu vermeiden.
       </p><p>
        Die Ranglistenwahl wird bei echten Wahlen verwendet. Beispielsweise
        verwendet Australien Single Transferable Vote (STV, auch bekannt als
        IRV). Die von CIVS unterstützten Condorcet-Methoden sind jedoch
        besser als IRV bei der Identifizierung von Konsenskandidaten. IRV
        kann einen Kandidaten wählen, obwohl eine Mehrheit der Wähler
        jemand anderen bevorzugen würde, und
       <a href="http://bolson.org/~bolson/2009/20090303_burlington_vt_mayor.html">das ist bei echten Wahlen
        der Fall</a>. Im Gegensatz zu den meisten anderen Condorcet-Systemen
        verfügt CIVS auch über einen proportionalen Darstellungsmodus.
       </p>
    }
}

Proportional_rep_expl {
  en {
The proportional mode is useful only for multiwinner elections, where you want to make sure that the voters' desires are expressed proportionally. Otherwise a majority of voters can decide all of the winners. Proportional mode is much slower than the regular nonproportional mode, and should not be used in cases where there are a lot of choices and a lot of winners. In most cases, but not all, the results from the proportional and nonproportional mode agree.  Click the 'more information' link to find out more about how this mode works.
  }
  de {
    Der proportionale Modus ist nur für Multiwinner-Wahlen sinnvoll, bei denen Sie sicherstellen möchten, dass die Wünsche der Wähler proportional ausgedrückt werden. Andernfalls kann eine Mehrheit der Wähler alle Gewinner bestimmen. Der proportionale Modus ist viel langsamer als der normale nicht-proportionale Modus und sollte nicht in Fällen verwendet werden, in denen es viele Auswahlmöglichkeiten und viele Gewinner gibt. In den meisten Fällen, aber nicht in allen Fällen, stimmen die Ergebnisse aus dem proportionalen und dem nicht-proportionalen Modus überein. Klicken Sie auf den Link „Weitere Informationen“, um mehr über die Funktionsweise dieses Modus zu erfahren.
  }
}
